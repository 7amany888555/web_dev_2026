What GitHub Pages Is

GitHub Pages hosts static websites directly from a repository.

When to Use It

  • GitHub Pages is useful for small websites like portfolios and class projects.

  • It works well when your site is built with static files such as HTML and images.

  • It creates a public link that you can share with instructors or employers.

How I Enabled It

Basic Steps

  1. I pushed my files to the main branch and opened the repository settings.

  2. In the Pages section, I selected the main branch to publish the site.

  3. After deployment, I tested navigation links to confirm all pages worked.
