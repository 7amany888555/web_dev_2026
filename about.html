What GitHub Does

Version Control in Practice

Repositories

A repo stores code, files, and a timeline of changes.

Why Repos Matter

  • Repositories keep a project organized in one place, including folders and files.

  • They also store history, so you can see what changed and when it changed.

  • This is useful for collaboration because multiple contributors can work safely.

Commits

Saving Progress

  • A commit is a saved checkpoint that records what changed and includes a message.

  • Good commit messages make it easier to understand the project history later.

  • When a bug appears, commits help you find where the issue started.
